# Contributing

Thank you for your interest in Diraigent!

## Code Contributions

This project does **not** accept external code contributions (pull requests). The codebase is maintained exclusively by the Zivue team.

## How You Can Help

We welcome:

- **Bug reports** — please [open an issue](../../issues/new?template=bug_report.md) with steps to reproduce
- **Feature requests** — tell us what you'd like to see via a [feature request](../../issues/new?template=feature_request.md)
- **Questions and discussion** — use the issue tracker for general questions

## Reporting Security Issues

If you discover a security vulnerability, please **do not** open a public issue. Instead, contact us directly at security@zivue.com.
